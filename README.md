Mise en page Flexible** : Utilisation de Flexbox pour l'alignement des éléments.
Design Responsive : 
  -Version Desktop: Affichage en deux colonnes (Profil à gauche, Formulaire à droite).
  - Version Mobile: Adaptation automatique en une seule colonne pour une lecture optimale sur smartphone.
Interface Moderne: 
  - Image de profil circulaire avec gestion du ratio (`object-fit`).
  - Effets visuels d'ombres portées (`box-shadow`) pour un aspect "carte".
  - États d'interaction (:focus) sur les champs de saisie et les boutons.
